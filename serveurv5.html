Serveur¶

class serveurv5.Server[source]¶

Bases : object

Classe représentant le serveur de l’application.

Cette classe gère les connexions des clients, la communication avec la base de données, et l’exécution des commandes envoyées par les clients.

creerChaineEvenement(premierX: int, deuxiemeX: int, troisiemeX: int, listeY, listeZ)[source]¶

Crée une chaîne d’événements formatée pour la fonction journalisation.

Paramètres:

  • premierX (int) – Type d’évènement : 0 Ajout ; 1 Modification ; 2 Suppression.

  • deuxiemeX (int) – Objet affecté par l’évènement : 0 Utilisateur ; 1 Tache ; 2 Sous tâche ; 3 Liste ; 4 Group ; 5 Groupes_utilisateurs ; 6 Etiquettes ; 7 Etiquettes_elements

  • troisiemeX (int) – Id de l’entrée affectée.

  • listeY (list | str) – Liste de noms ou d’indices de colonnes, selon le cas.

  • listeZ (list) – Liste des valeurs modifiées dans l’ordre des N° de colonne.

Lève:

  • pymysql.MySQLError – Erreur lors de la connexion à la base de données.

  • ValueError – Si certaines colonnes dans listeY ne correspondent pas.

Renvoie:

Chaîne d’événements formatée.

Type renvoyé:

str | None
